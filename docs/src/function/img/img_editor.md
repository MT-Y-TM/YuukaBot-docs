# 图片简单编辑

:::tip
由于腾讯的要求，本功能在 `QQ官方` 平台上暂不可用
:::

## 概述

本功能提供了对图片进行简单变换的指令

## 指令

:::tip
图片参数可以跟在指令后，也可以分开发送——先发送指令后发送图片，也可以通过回复消息来当做参数
:::

### 水平翻转

- 指令： `水平翻转`

- 参数：

  - `图片`： 需要操作的图片

- 详情：

  将图片水平翻转
