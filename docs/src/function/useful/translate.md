# 翻译

## 概述

本功能提供了翻译功能

## 指令

### 文字翻译

- 指令： `[源语言]译[目标语言][原文]`

- 参数：

  - `源语言`： 原来的语言，详见 [支持语言](#支持语言)
  - `目标语言`： 要翻译的语言，详见 [支持语言](#支持语言)
  - `原文`： 要翻译的东西

- 详情：

  从 `源语言` 翻译为 `目标语言`

- 使用示例：
  - 中译英这是什么地方啊
  - 自译自What`s up

### 图片翻译

- 指令： `图片[源语言]译[目标语言]`

- 参数：

  - `源语言`： 原来的语言，详见 [支持语言](#支持语言)
  - `目标语言`： 要翻译的语言，详见 [支持语言](#支持语言)

- 详情：

  将图片从 `源语言` 翻译为 `目标语言`
