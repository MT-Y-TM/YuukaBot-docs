# 天气

## 概述

本功能提供了有关 **天气** 的功能

## 指令

### 查天气

- 指令：
  - `天气[地区](*上级地区)`
  - `[地区](*上级地区)天气`

- 参数：

  - `地区`： 要查询的地区
  - `上级地区`： 可选参数，用于筛选同名地区

- 详情：

  查询天气

- 使用示例：

  - `哈尔滨天气`： 查询 `哈尔滨` 天气
  - `思明*厦门天气`： 查询 `厦门` 下辖的 `思明` 天气
  - `天气西安`： 查询 `西安` 天气
