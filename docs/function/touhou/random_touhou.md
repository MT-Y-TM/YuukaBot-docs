# 随机东方

## 指令

- 指令： `随机东方(-w):(英文tag)`

- 参数：

  - `-w`： 可选标识，不填获取横图，填获取竖图
  - `英文tag`： 可选参数，指定所需 tag

- 详情：

  英文 tag 可填 人物英文名、物品英文名等

- 使用示例：

  - `随机东方`： 表示获取随机的东方横屏图
  - `随机东方-w`： 表示获取随机的东方竖屏图
  - `随机东方:yuyuko`： 表示获取随机的横屏幽幽子图
  - `随机东方-w:renko`： 表示获取随机的竖屏莲子图
