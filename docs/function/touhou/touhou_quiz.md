# 东方问答

## 原曲认知

### 开始原曲认知

- 指令： `原曲认知(范围名)`

- 参数：

  - `范围名`：可选，可用 [**查看原曲范围列表**](/function/touhou/touhou_quiz.md#查看原曲范围列表) 获取

### 设置原曲范围 <Badge type="tip" text="群管或群主" vertical="top" />

- 指令： `设置原曲范围[范围名]`

- 参数：

  - `范围名`：可用 [**查看原曲范围列表**](/function/touhou/touhou_quiz.md#查看原曲范围列表) 获取

- 详情：

    设置本群原曲范围

### 查看原曲范围列表

- 指令： `查看原曲范围列表`

- 详情：

    查看所有原曲范围

### 回答原曲认知

- 指令： `*这是[答案]`

- 参数：

  - `答案`：答案选项
