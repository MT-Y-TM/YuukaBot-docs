# 语音相关

## 概述

本功能提供了有关语音的功能

## 指令

### 给我讲

- 指令： `给我讲[句子]`

- 参数：

  - `句子`： 要讲的句子

- 详情：

  中文语音

### 日语

- 指令： `日语[句子]`

- 参数：

  - `句子`： 要讲的句子

- 详情：

  日文语音

### 油库里

- 指令： `油库里[句子]`

- 参数：

  - `句子`： 要讲的句子

- 详情：

  油库里语音

:::tip
`日语`、 `油库里` 功能不自带翻译，应输入日文与日本汉字
:::

### 德语

- 指令： `德语[句子]`

- 参数：

  - `句子`： 要讲的句子，不自带翻译

- 详情：

  德语语音


### 支付宝到账

- 指令： `支付宝到账[钱数]`

- 参数：

  - `钱数`： 模拟的钱数

- 详情：

  支付宝到账语音
