# 搜索功能

## 概述

本功能提供了搜索功能

## 指令

### 搜索

- 指令： `搜索[关键词]`

- 参数：

  - `关键词`： 搜索关键词

- 详情：

  百度搜索

### 文库搜索

- 参数：

  - `关键词`： 搜索关键词

- 详情：

  百度文库搜索

### 谷歌搜索

- 指令： `谷歌搜索(-参数)[关键词]`

- 参数：

  - `-参数`： 可选参数，-1、-2、-3为镜像站，-e为源站
  - `关键词`： 搜索关键词

  谷歌搜索
