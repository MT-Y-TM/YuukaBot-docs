# 100%鲜橙汁

## 概述

本功能提供了一系列有关 `100% 鲜橙汁` 的功能

## 指令

### 今日橙汁

- 指令： `/今日橙汁`

- 别名： `今日橙汁`, `今日百橙`, `/今日百橙`

- 详情：

  来看看你今天的橙汁运势吧！

### 橙汁查卡

- 指令： `/橙汁查卡 [卡片名]`

- 别名： `/百橙查卡`

- 参数：

  - `卡片名`： 卡片名，支持模糊搜索

- 详情：

  橙汁查卡/查角色

### 橙汁抽卡

- 指令： `/橙汁抽卡 [类别]`

- 别名： `/百橙抽卡 [类别]`

- 参数：

  - `类别`： 全部/boss技能/普通/hyper/其它/玩家/NPC

- 详情：

  橙汁随机抽卡！
