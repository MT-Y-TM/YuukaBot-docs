# Hypixel 查询

## 概述

本功能提供了有关 Hypixel 的相关功能

## 指令

:::tip
可用 [/hyp bind](#hyp-bind) 绑定账户，来省去每次都要输账号的麻烦
:::

### /hyp

- 指令： `/hyp (游戏名)`

- 参数：

  - `游戏名`： 游戏内名字，若绑定了账户则可选

- 详情：

  查看 hypixel 基本数据

### /hyp bind

- 指令： `/hyp bind [游戏名]`

- 参数：

  - `游戏名`： 游戏内名字

- 详情：

  绑定账号
